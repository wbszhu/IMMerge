# IMMerge Documentation

This directory contains the full IMMerge user manual. For installation
and a one-command quick start, see the [project README](../README.md).

## Contents

- [Usage Guide](usage.md) — full CLI reference for `merge_files.py` and
  `make_info.py`, input file format, Python module API, additional
  examples.
- [Algorithm Details](algorithm.md) — how IMMerge combines imputation
  quality (R²) across input files using Fisher z-transformation and
  weighted average, plus combined MAF and allele frequency.
